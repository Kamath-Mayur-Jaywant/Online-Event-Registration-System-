# Online Event Registration System

## Overview
The **Online Event Registration System** is a web-based platform that allows users to browse, register, and pay for events online. Admins can create and manage events, while users can view events, register, and receive email confirmations. The system is designed to be scalable, reliable, and user-friendly.

---

## Features
- User registration and authentication
- Event creation, update, and management (Admin)
- Event browsing and search
- Event registration with online payment
- Admin dashboard for monitoring events and registrations
- Email notifications for registrations and payments
