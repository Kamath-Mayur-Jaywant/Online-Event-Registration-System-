# Online Event Registration System - Frontend

## Overview
This is the frontend of the Online Event Registration System. It provides a user-friendly interface for users to browse events, register, make payments, and for admins to manage events.

## Features
- User registration and login
- Event browsing and search
- Event registration with online payment
- Admin dashboard for event management
- Responsive UI design

## Technology Stack
- **Frontend Framework**: React / Vue / Angular / Vanilla HTML, CSS
